COMO ATUALIZAR O GITHUB

1. Extraia o ZIP de atualização.
2. Abra o repositório do DAKKAI no GitHub.
3. Clique em Add file > Upload files.
4. Envie todos os arquivos e confirme a substituição.
5. Clique em Commit changes.
6. Aguarde o deploy da Vercel.
7. No computador, pressione Ctrl + F5 para limpar a versão antiga.
